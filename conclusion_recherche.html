Conclusion de la recherche

Pour arriver ou nous en sommes dans ce projet, nous avons dû faire beaucoup de recherches. Pour effectuer ces recherches, nous ne nous sommes pas tournés vers les livres ou les professeurs ou nos notes de cours, nous avons utilisé le moteur de recherche de Google. Sans Google, nous n'aurions pas pu accomplir tout cela. Grâce à cet outil, nous avons pu trouver des sites de tutoriel pour les débutants, nous avons pu trouver des exemples sur lesquels nous nous sommes fiés, nous avons pu avoir accès à la documentation des technologies que nous avons utilisé et, bien sûr, nous avons pu trouver des renseignements sur les erreurs que nous rencontrions. Donc, savoir bien utiliser un moteur de recherche nous a permis d'atteindre nos objectifs. Selon nous, les prochaines investigations sur notre application, si elle était à continuer, porteraient sur le CSS pour rendre le menu un peu plus attrayant. Aussi, nous voudrions que l'utilisateur puisse être capable changer le raccourci clavier (pour l'instant 'Shift+z') dans les options de l'extension pour plus de flexibilité.
